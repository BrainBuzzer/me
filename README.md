## How to setup for development:

1. Clone the repo

``
git clone <repo>
``


(Optional) use a virtualenv

``
virtualenv venv
``


``
source venv/bin/activate
``


2. Install dependencies

``
pip3 install -r requirements.txt
``

``
npm install
``


3. Start the server
``
./manage.py runserver
``
You can login @ 127.0.0.1:8000/accounts/login
